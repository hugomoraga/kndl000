```javascript
// Meditación void
function vacio() {
  // El vacío no necesita ser llenado
  // La ausencia es una forma de presencia
  return undefined;
}

// ¿Qué retorna una función que no retorna?
const nada = vacio();
console.log("Resultado de vacio():", nada);

// Pero el acto de llamarla existe
// La ejecución deja rastro
// El vacío tiene forma
console.log("Tipo de nada:", typeof nada);
console.log("¿Es nada igual a undefined?", nada === undefined);
```

---

_Una función que no hace nada, pero existe. El código como meditación sobre la ausencia._

```python
# Condicional existencial
if True:
    print("Soy verdadero")
else:
    print("Nunca me ejecutaré")

# ¿Qué es verdadero?
# ¿Qué es falso?
# ¿La verdad es absoluta o relativa?

if False:
    print("Nunca existiré")
else:
    print("Soy la alternativa")

# El código que nunca se ejecuta
# ¿Existe o no existe?
# ¿El código muerto es código?
# ¿La posibilidad no realizada es real?
```

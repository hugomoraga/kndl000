```python
def infinito():
    """
    Una función que se llama a sí misma
    infinitamente. ¿Es esto un error o una
    declaración filosófica?
    """
    print("Estoy aquí")
    infinito()  # Se llama a sí misma

# Nunca ejecutes esto
# O sí, si quieres contemplar el infinito
# hasta que el stack overflow te devuelva
# a la realidad finita
```

---

_La recursión infinita como metáfora del eterno retorno. El código que busca su propio origen._
